# Logos dos Clientes

Coloque os logos aqui no formato:
- `<slug>.png` (recomendado, PNG transparente)
- `<slug>.svg` (vetorial, melhor)

Tamanho ideal: 256x256px ou maior, quadrado, fundo transparente.

## Atuais
- `estacionepark.png` - logo de estacionamento + aviao (azul)
- `masteraviacao.png` - (pendente)
- `alessandro.png` - (pendente)

## Como o admin usa
No CLIENTES_CRM (admin/index.html), cada cliente pode ter:
`logoUrl: '/clientes/_logos/<slug>.png'`
